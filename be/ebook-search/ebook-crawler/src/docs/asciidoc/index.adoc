== 공통 오류 응답

모든 오류 응답은 다음과 같은 구조를 가집니다:

[source,json]
----
{
  "status": "HTTP 상태 코드와 설명",
  "message": "에러 메시지",
  "timestamp": "에러 발생 시각"
}
----

=== 필드 설명

* `status`: HTTP 상태 코드와 그에 대한 설명
* `message`: 구체적인 에러 메시지
* `timestamp`: 에러가 발생한 시각 (ISO 8601 형식)
